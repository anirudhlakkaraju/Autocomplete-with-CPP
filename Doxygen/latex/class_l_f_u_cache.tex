{\footnotesize\ttfamily void L\+F\+U\+Cache\+::put (\begin{DoxyParamCaption}\item[{string}]{word }\end{DoxyParamCaption})}

Inserts word into the cache and updates its frequency. ~\newline
If word already exists in the cache then simply increments its frequency. ~\newline
If word does not exist in the cache, then two cases arise -\/
\begin{DoxyEnumerate}
\item The max capacity is reached -\/ Ejects word(s) with min\+Freq to make space and then inserts given word.
\item The max capacity is not reached -\/ Simply inserts the given word which now has min\+Freq.
\end{DoxyEnumerate}
